Changes to the privacy policy

This privacy policy may be updated at any time to reflect how Utilityweb gathers and proccesses user information. Utilityweb does not record user's contact details unless for the purpose of communicating about an enquiry(such as when a user contacts Utilityweb via email). As such, Utilityweb cannot email users to notify of changes to the privacy policy. Users are advised to periodically check the privacy policy for updates.

Information that is collected

Google Analytics:

Utilityweb uses google analytics, which collects user information. Examples of information collected include: IP addresses; session durations; pages visited; reffering details(how you got to the site); browser and operating system. Additionally, Utilityweb uses custom user flow tracking on some pages through Google Analytics, which is used to determine user interactions with the website with the purpose of improving user experience (Such as moving frequently used elements to improve work flow). For more information about how google uses collected information and how you can control the information collected of you, visit https://policies.google.com/technologies/partner-sites

How Utilityweb uses collected information.

Information collected through Google Analytics is used by Utilityweb with the sole purpose of improving the website and user experience.
Google may additionally use analytic information to serve targeted ads, depending on user settings. For more information about how Google uses analytics information, visit policies.google.com/technologies/partner-sites

Cookies

Google Analytics uses cookies to identify visistors. For information about how Google uses cookies, see policies.google.com/technologies/cookies

Utilityweb uses cookies on some of its pages to improve user experience. Examples of such uses include storing utility settings in your browser, so that they are remembered on your next session.
